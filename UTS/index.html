UJIAN TENGAH SEMESTER PRAKTIKUM SIG WEB


Ringkasan Mengenai SIG WEB

Mata Kuliah Praktikum Sistem Informasi Geografi Berbasis web merupakan mata kuliah yang mempelajari menggabungkan peta dengan web interaktif. SIG Web merupakan salah satu bentuk untuk menampilkan peta dalam website agar banyak pengguna dapat mengakses. Dalam SIG Web dapat mengumpulkan, mengelola, menganalisis, dan memvisualisasikan data spasial melalui web yang telah dibuat.Berikut berisi kegiatan praktikum mulai dari minggu 1 sampai dengan minggu 7.


Minggu ke-1 - HTML, CSS, dan WIX

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar dari sebuah halaman web. HTML digunakan untuk menentukan elemen-elemen pada halaman web seperti teks, gambar, link, dan sebagainya.

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan styling elemen-elemen yang telah ditentukan dalam HTML. Dengan CSS, pengembang web memungkinkan untuk mengontrol warna, font, ukuran, dan elemen desain lainnya pada halaman web. CSS membuat perubahan dalam tampilan halaman web menjadi lebih fleksibel dan mudah dikelola, karena pengguna dapat mengubah styling secara terpusat tanpa harus mengedit setiap elemen di halaman tersebut.

Wix adalah sebuah platform pembuatan web yang membantu pengguna untuk membuat situs web sendiri tanpa harus memiliki pengetahuan mendalam tentang pemrograman atau desain web.

Pada pelaksanaan praktikum materi ini, dilakukan dengan membuat sebuah WEB dengan HTML dan CSS sederhana. Selain itu, dilakukan pembuatan Curiculum Vitae (CV) dengan menggunakan website WIX.


Minggu ke-2 - JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk mengembangkan situs web interaktif dan dinamis. Javascript merupakan salah satu bahasa pemrograman yang paling umum digunakan di seluruh dunia untuk pengembangan web. JavaScript memiliki peran penting dalam menjadikan halaman web lebih responsif, berfungsi, dan interaktif.

Pada praktikum yang telah dilakukan, dilakukan pembuatan script menggunakan java script untuk menjawab lima soal yang telah diberikan.


Minggu ke-3 & ke-5 - Bootstrap Framework

Bootstrap adalah sebuah framework front-end sumber terbuka yang digunakan untuk memudahkan pengembangan situs web dan aplikasi web responsif (responsive web) yang menarik dan berfungsi dengan baik. Framework ini dibuat dengan bahasa markup CSS yang membantu pengguna dalam pembuatan web. Bootstrap menyediakan serangkaian alat, gaya, dan komponen yang siap pakai untuk mempercepat proses pengembangan web.

Pada praktikum yang telah dilakukan, pertama mengenal framework bootstrap yang akan digunakan. Lalu menggunakan Bootstrap dalam pembuatan CV yang berisi biodata pribadi pemilik web serta link tugas yang telah dikerjakan selama praktikum SIG WEB


Minggu ke-6 - Leaflet JS beserta Pluginnya

Leaflet JS adalah sebuah perpustakaan JavaScript open-source yang digunakan untuk membuat peta interaktif di situs web. Leaflet dirancang untuk menjadi ringan, cepat, mudah digunakan, dan memiliki fokus pada responsif dan kustomisasi. Leaflet merupakan salah satu perpustakaan peta yang paling populer di antara pengembang web dan digunakan secara luas dalam proyek-proyek yang memerlukan integrasi peta interaktif.

Pada praktikum, telah dilakukan pembuatan peta interaktif menggunakan Leaflet JS dengan menambahkan beberapa plugin seperti plugin geolocating dan routing


Minggu ke-7 - Leaflet JS dengan GeoJSON

Pada pelaksanaan praktikum materi ini, melanjutkan pembuatan peta dengan Leaflet JS dengan menambahkan shapefile. Shapefile yang dibuat dalam QGIS atau ArcGIS akan di save dalam bentuk GeoJSON. File GeoJSON akan dimasukkan dalam script yang nanti akan muncul dalam peta interaktif yang telah dibuat.
